# 任务管理应用 (Task Manager) - 设计文档

## 应用概述

任务管理应用是一款帮助用户高效组织和跟踪日常任务的移动应用。用户可以创建任务、设置优先级、标记完成状态，并按类别管理任务。应用采用本地存储方案，确保快速响应和离线可用性。

---

## 屏幕列表

### 1. **首页 (Home Screen)**
- **功能**：显示所有任务列表，支持按状态筛选
- **内容**：
  - 顶部统计卡片（待完成任务数、已完成任务数、总任务数）
  - 任务列表（支持滚动）
  - 浮动操作按钮 (FAB) - 创建新任务
  - 底部标签栏导航

### 2. **任务详情屏幕 (Task Detail Screen)**
- **功能**：查看和编辑单个任务的详细信息
- **内容**：
  - 任务标题
  - 任务描述
  - 优先级选择（高、中、低）
  - 截止日期选择
  - 任务分类标签
  - 完成状态切换开关
  - 编辑和删除按钮

### 3. **创建/编辑任务屏幕 (Create/Edit Task Screen)**
- **功能**：创建新任务或编辑现有任务
- **内容**：
  - 标题输入框
  - 描述输入框
  - 优先级单选按钮（高、中、低）
  - 日期选择器（截止日期）
  - 分类选择（下拉菜单）
  - 保存和取消按钮

### 4. **分类管理屏幕 (Categories Screen)**
- **功能**：管理任务分类
- **内容**：
  - 分类列表
  - 每个分类的任务计数
  - 添加新分类按钮
  - 删除分类功能

### 5. **设置屏幕 (Settings Screen)**
- **功能**：应用设置和个性化
- **内容**：
  - 主题切换（浅色/深色模式）
  - 通知设置
  - 数据导出/导入
  - 关于应用

## 主要用户流程

### 流程 1：创建任务
1. 用户在首页点击浮动操作按钮 (FAB)
2. 进入创建任务屏幕
3. 填写任务标题、描述、优先级、截止日期和分类
4. 点击保存按钮
5. 返回首页，新任务显示在列表顶部

### 流程 2：完成任务
1. 用户在首页点击任务卡片
2. 进入任务详情屏幕
3. 点击完成状态开关
4. 任务标记为已完成（视觉反馈：删除线、灰显）
5. 返回首页

### 流程 3：编辑任务
1. 用户在任务详情屏幕点击编辑按钮
2. 进入编辑任务屏幕
3. 修改任务信息
4. 点击保存按钮
5. 返回任务详情屏幕

### 流程 4：删除任务
1. 用户在任务详情屏幕点击删除按钮
2. 显示确认对话框
3. 用户确认删除
4. 任务从列表中移除，返回首页

### 流程 5：按分类筛选
1. 用户在首页点击分类标签栏
2. 选择特定分类
3. 列表更新，仅显示该分类的任务
4. 点击"全部"恢复显示所有任务

## 主要内容和功能

### 任务卡片 (Task Card)
- **显示内容**：
  - 任务标题
  - 优先级指示器（颜色编码）
  - 截止日期（如果设置）
  - 分类标签
  - 完成状态复选框
- **交互**：点击进入详情，长按显示快速操作菜单

### 统计卡片 (Stats Card)
- **显示内容**：
  - 待完成任务数
  - 已完成任务数
  - 总任务数
- **位置**：首页顶部，使用三列布局

### 优先级指示器
- **高优先级**：红色 (#EF4444)
- **中优先级**：黄色 (#F59E0B)
- **低优先级**：绿色 (#22C55E)

## 设计原则

1. **移动优先**：所有设计基于竖屏 (9:16) 和单手操作
2. **iOS 风格**：遵循 Apple Human Interface Guidelines，确保应用感觉像原生 iOS 应用
3. **清晰的视觉层级**：使用大小、颜色和间距区分内容优先级
4. **一致的交互**：所有按钮和输入框遵循相同的交互模式
5. **快速反馈**：用户操作立即获得视觉反馈（按钮按下、加载指示器等）
6. **无障碍性**：确保文本大小足够大，颜色对比度充分

## 数据模型

### Task (任务)
```typescript
{
  id: string;              // 唯一标识符
  title: string;           // 任务标题
  description: string;     // 任务描述
  priority: 'high' | 'medium' | 'low';  // 优先级
  completed: boolean;      // 完成状态
  dueDate: Date | null;    // 截止日期
  category: string;        // 分类
  createdAt: Date;         // 创建时间
  updatedAt: Date;         // 更新时间
}
```

### Category (分类)
```typescript
{
  id: string;              // 唯一标识符
  name: string;            // 分类名称
  color: string;           // 分类颜色
  createdAt: Date;         // 创建时间
}
```

## 技术栈

- **框架**：React Native (Expo)
- **UI 框架**：NativeWind (Tailwind CSS)
- **状态管理**：React Context + AsyncStorage
- **数据存储**：AsyncStorage (本地存储)
- **导航**：Expo Router
- **类型检查**：TypeScript
